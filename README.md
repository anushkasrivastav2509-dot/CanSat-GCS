# 🚀 CanSat Ground Control Station (GCS)

A professional Ground Control Software (GCS) developed for a CanSat mission.

## Features

- 📡 Live Telemetry Dashboard
- 🛰 3D CanSat Orientation
- 📍 GPS Tracking
- 📈 Real-Time Charts
- 🎮 Mission Control Panel
- 💻 Mission Console
- 🔋 Battery Monitoring

## Technologies

- HTML5
- CSS3
- JavaScript
- Chart.js
- Three.js
- Leaflet.js
- Git & GitHub
